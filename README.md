# Dream House
Dream House adalah sebuah aplikasi mobile berbasis Android yang dirancang sebagai platform 
pencarian dan eksplorasi properti modern. Aplikasi ini bertujuan untuk memudahkan pengguna 
dalam menemukan hunian sesuai kebutuhan baik rumah, apartemen, villa, maupun properti 
komersial lainnya.
